STRATOSPHERE

A third-person gameplay demo
What is so special about this third-person gameplay?
This video game demo uses root-motion for the character movement, along with other unity techniques to improve the overall gameplay quality.

Root-Motion

Instead of using scripts to translate an object you can use the actual object animator to move the object.

Cinemachine

It's a powerful camera controller in unity. I use it for creating different cameras and transitions.

Post-processing

Using linear color space along with tonemapping can improve the overall look of the game.
